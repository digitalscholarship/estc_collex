Coming Soon

Visualization Tool for Poetic Elements 1750-1850

This web-based tool written in Java will allow users to try out various hypotheses about poetry production during this period.  From a drop-down box listing twenty items, users will be able to select as many elements as they wish to "privilege" or to view as part of their visual model.  These elements will range from metrical forms to semantics, publication venue to graphics on the page, images, bookboards, slipcases . . . .

New to the Poetess Archive

Rather than a static bibliographic list, each entry has been put into a database which will allow you myriad ways of searching and ordering the information available at this site. We do also have some full texts available here corresponding to bibliographic entries, and so the database will return links to full-text html and xml pages, as well as to pages that provide only bibliographic information. We hope to acquire many, many more full texts soon; please return often to see our progress.
